# Car-Crash-Data-Analysis-Project

Performed end‑to‑end data analysis on a car crash dataset (2003–2015) to identify key factors influencing road accidents.

Conducted extensive data cleaning: handling missing values, validating coordinates, removing noise, mapping seasons and time periods, and preprocessing text using stopwords removal and tokenization.

Analyzed accident patterns by season, time of day, day of week, collision type, and primary contributing factors.

Generated visual insights using heatmaps, pie charts, bar charts, scatter plots, and time‑series trends to highlight accident frequency and major causes.

Tools: Python, Pandas, NumPy, NLTK, Seaborn, Matplotlib.

Achieved: Identified the most common accident causes (e.g., Failure to Yield, Following Too Closely), peak accident times, and seasonal patterns to support road‑safety decision making.
